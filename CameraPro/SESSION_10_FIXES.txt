```markdown
# Camera Pro

Advanced Windows Camera Application

## Features
- Live camera preview
- Photo capture (JPEG, PNG, BMP)
- Video recording (MP4)
- Real-time filters
- Camera controls (exposure, focus, white balance)
- Multi-camera support
- Storage organization

## Requirements
- Windows 10 (version 1903 or later)
- .NET 8 Runtime (included in self-contained build)

## Installation
1. Run CameraPro.exe from the publish folder

## Usage
- Space: Capture photo
- R: Start/stop recording
- P: Pause recording
- F11: Fullscreen
- G: Toggle grid
- 1-9: Quick filters

## Storage
Photos: C:\Users\[User]\Pictures\CameraPro
Videos: C:\Users\[User]\Videos\CameraPro
```
